# v1.0 备份说明

这是 v1.0 版本的完整备份，保存了基础点击控制功能。

## 如何恢复到 v1.0

如果需要恢复到这个版本，复制以下文件：

### 核心文件
- `src/main.js` - 主应用逻辑
- `src/managers/SimpleLightManager.js` - 灯光管理器
- `src/config/light-config.js` - 灯光配置
- `index.html` - HTML 入口

### 特性
- 点击模型控制灯光
- 时间模式自动切换
- 18个灯光覆盖所有房间

## 当前文件状态

所有文件已保存在项目根目录，可以随时查看或恢复。
